LLM Output Diversity Study

Study Overview

Large Language Models (LLMs) are stochastic—they don't produce the same output every time. When you give an LLM the same prompt multiple times, you get a distribution of possible outputs. Higher "temperature" settings typically produce more diverse outputs, while lower temperatures produce more similar ones.

In this study, you'll compare two sets of outputs from the same prompt but generated with different temperature settings. Your task is to determine which set has higher diversity (orange vs. blue). You'll do this using two different visualization interfaces.

  • You'll see a link that opens a visualization showing both sets side-by-side.
  • After viewing, select whether orange or blue has higher diversity.
  • You'll have 3 minutes per interface to answer as many comparison questions as you can.
